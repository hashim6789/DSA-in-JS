# Data Structures and Algorithms in JavaScript

Welcome to my Data Structures and Algorithms (DSA) repository! This project is a culmination of my journey through the fundamentals and advanced concepts of DSA in JavaScript, spread over three weeks of intense learning and coding.

## Week 1: Basics of Data Structures
**Content Covered**:
- **Introduction to Data Structures**: Understanding what data structures are and their importance.
- **Arrays**: Implementation, basic operations, and use cases.
- **Linked Lists**: Single and double linked lists, basic operations.
- **Strings**: Manipulation, common operations, and algorithms.
- **Practice Tasks**: Sample operations and tasks related to these topics.

**Key Folders**:
- `arrays`: Implementations and operations related to arrays.
- `linkedLists`: Code for single and double linked lists.
- `strings`: String manipulation and common algorithms.

## Week 2: Intermediate Data Structures and Sorting Algorithms
**Content Covered**:
- **Stacks**: LIFO structure, basic operations, and use cases.
- **Queues**: FIFO structure, including circular queues.
- **Hash Tables**: Implementation, collision handling, and use cases.
- **Sorting Algorithms**: 
  - **Bubble Sort**
  - **Insertion Sort**
  - **Selection Sort**
  - **Merge Sort**
  - **Quick Sort**

**Key Folders**:
- `stack`: Implementation and operations for stacks.
- `queue`: Code for different types of queues.
- `HashTable`: Hash table implementation and collision handling.
- `Algorithms`: Various sorting algorithms implemented and explained.
